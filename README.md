# 板球学习网站

## 项目概述

基于完整的设计规范和研究成果开发的专业的板球学习网站，参考W3Cschool风格，具有简洁美观的界面和良好的用户体验。

## 核心功能

### ✅ 已实现功能

1. **左侧树形导航**
   - 展示7大分类的150+个板球知识点
   - 支持层级展开折叠
   - 清晰的视觉层级和图标标识
   - 难度标签（入门/进阶/高级）

2. **右侧内容展示**
   - 显示选中知识点的详细内容
   - 支持图文并茂展示
   - 面包屑导航
   - Markdown格式内容渲染

3. **响应式设计**
   - 适配桌面、平板、移动端
   - 移动端汉堡菜单
   - 流畅的交互体验

4. **搜索功能**
   - 顶部搜索框
   - 支持知识点搜索
   - 实时搜索建议

5. **简洁界面**
   - 深色主题，避免大色块
   - 专业学习氛围
   - 基于设计Tokens的视觉系统

## 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式系统**: Tailwind CSS
- **路由管理**: React Router
- **图标库**: Lucide React
- **UI组件**: shadcn/ui
- **状态管理**: React Hooks

## 设计系统

### 颜色系统
- **主色调**: Primary Blue (#2196f3)
- **背景色**: 深色主题 (#212121)

## 知识体系结构

### 7大分类
1. **基础知识** - 历史发展、概念术语、比赛形式、装备场地
2. **规则系统** - 比赛规则、判罚系统、特殊规则、争议案例
3. **技术技能** - 击球技术、投球技术、守备技术、跑动跑分
4. **战术策略** - 团队战术、个人战术、比赛管理、心理战术
5. **训练方法** - 基础训练、专项训练、训练计划、训练方法
6. **心理战术** - 心理技能、压力管理、情绪管理、沟通技巧
7. **比赛管理** - 赛前管理、赛中管理、赛后管理

### 150+个知识点
- **入门级**: 基础概念和简单应用
- **进阶级**: 深入技术和战术应用
- **专业级**: 高级策略和心理战术
